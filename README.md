## Kurulum

1. Dosyaları bilgisayarınıza indirin.
2. `index.html` dosyasını tarayıcıda açın.
3. GitHub Pages için tüm dosyaları public bir repoya yükleyin.

## GitHub Pages Yayınlama

1. GitHub hesabınızda `kumsal04.github.io` adında public repo oluşturun.
2. Bu klasördeki tüm dosyaları repoya yükleyin.
3. Settings > Pages bölümünden branch olarak `main` seçin.
4. Yayın adresi: https://kumsal04.github.io
