# Random User API Kullanımı

## Proje Açıklaması

JavaScript ile harici API'den veri çekme, kullanıcıları listeleme ve arayüzde detay gösterme projesi.

## Kullanılan Teknolojiler

HTML, CSS, JavaScript, API

## Özellikler

- API'den veri çekme
- Kullanıcı listeleme
- Filtreleme ve yenileme işlemleri
- Hata mesajı yönetimi
- Detay modal yapısı

## Kurulum

1. Repoyu indirin veya klonlayın.
2. Gerekli dosyaları ilgili geliştirme ortamında açın.
3. Projenin ana dosyasını çalıştırın.
4. Gerekirse ekran görüntüsü ve demo bağlantılarını README içerisine ekleyin.
