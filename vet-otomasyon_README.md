# Vet Otomasyon

## Proje Açıklaması

Veteriner kliniği için hasta, randevu ve takip süreçlerini yönetmeye yönelik otomasyon projesi.

## Kullanılan Teknolojiler

Python, tkinter, MySQL

## Özellikler

- Hasta kayıt yapısı
- Randevu takip mantığı
- Veteriner paneli
- Veri tabanı bağlantısı
- Kullanıcı dostu masaüstü arayüz

## Kurulum

1. Repoyu indirin veya klonlayın.
2. Gerekli dosyaları ilgili geliştirme ortamında açın.
3. Projenin ana dosyasını çalıştırın.
4. Gerekirse ekran görüntüsü ve demo bağlantılarını README içerisine ekleyin.
