# MiyHav Petshop & Sahiplendirme

## Proje Açıklaması

Petshop ürünleri, ücretsiz sahiplendirme sayfaları, filtreleme, favori ekleme ve responsive web arayüzü içeren kapsamlı web projesi.

## Kullanılan Teknolojiler

HTML, CSS, Bootstrap, JavaScript, JSON

## Özellikler

- Responsive ana sayfa ve ürün sayfaları
- Sahiplendirme ilanları
- Ürün listeleme ve filtreleme
- Favori ekleme mantığı
- GitHub üzerinden public proje paylaşımı

## Kurulum

1. Repoyu indirin veya klonlayın.
2. Gerekli dosyaları ilgili geliştirme ortamında açın.
3. Projenin ana dosyasını çalıştırın.
4. Gerekirse ekran görüntüsü ve demo bağlantılarını README içerisine ekleyin.
