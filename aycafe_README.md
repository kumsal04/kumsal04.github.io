# AyCafe Sipariş Otomasyonu

## Proje Açıklaması

Kafe sipariş süreçlerini yönetmek için hazırlanmış C# otomasyon projesi.

## Kullanılan Teknolojiler

C#

## Özellikler

- Sipariş akışı
- Ürün/sipariş yönetimi
- Masa veya müşteri odaklı takip mantığı
- Otomasyon arayüzü

## Kurulum

1. Repoyu indirin veya klonlayın.
2. Gerekli dosyaları ilgili geliştirme ortamında açın.
3. Projenin ana dosyasını çalıştırın.
4. Gerekirse ekran görüntüsü ve demo bağlantılarını README içerisine ekleyin.
